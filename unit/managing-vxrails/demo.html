Pivotal Ready Architecture

Goal

Understand the benefits of VxRail and VxRail Manager for deployment of the Pivotal Ready Architecture.

Prerequisites

  • VxRail Deployed
  • Access & ID to http://democenter.dell.com
  • Access to Live VxRail environment for observability demo of VxRail Manager

    VxRail Overview:

    1. This demo will give the customer an overview of the most important functions of VxRail and show how it is a key buidling block for the most reliable, repeatable way to stand up Pivotal Cloud Foundry On-Premesis.

    Install VxRail:

    1. Point your browser to https://democenter.dell.com/
    2. Create an Account and Login
    3. In the search box type VxRail and find VxRail 4.5 and click Connect
    4. Click Start click Install VxRail.
    5. Click Get Started then accept the EULA and click Accept.
    6. Click Confirm to accept the node configuration and click Next.
    7. Select the JSON Configuration then click the file and Open then Next.
    8. Click Get Started then accept the EULA and click Accept.
    9. Now you will see all the values populated for build. Click Next.
    10. Scroll to the bottom of the page and view the configuration settings. Click Continue
    11. Configure the vMotion Network and click Next.
    12. Configure vSAN and click Next.
    13. Configure VM Networks and click Next.
    14. Configure Solutions for logging (in this installation we have used log insight) and click Next.
    15. Configure all of the login credentials and click Next.
    16. Validate that all of tasks have been completed and all checks are green and click Validate.
    17. Any errors would appear on this screen. Now download the JSON file to review the config, then click Build VxRail then Start Configuration.
    18. After build completes your ready to log into the VxRail appliance
    19. Click ESC to return to the next demo.
